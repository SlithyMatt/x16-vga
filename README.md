# x16-vga
VGA-style demo for Commander X16

# Build and run demo
In bash (use Git bash shell if on Windows):
<pre>
$ ./build_tools.sh
$ ./build_bins.sh brillig.data
$ /path/to/x16emu -bas vga.bas -run
</pre>
